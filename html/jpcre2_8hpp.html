Detailed Description

Main header file for JPCRE2 library to be included by programs that uses its functionalities.

It includes the pcre2.h header, therefore you shouldn't include pcre2.h separately in your program. Make sure to link PCRE2 library when compiling.

Macro Definition Documentation

§ JPCRE2_DISABLE_CODE_UNIT_WIDTH_VALIDATION

#define JPCRE2_DISABLE_CODE_UNIT_WIDTH_VALIDATION

By default JPCRE2 checks if the code unit width equals to sizeof(Char_T)*CHAR_BIT, if not,it will produce compile error.

This check can be disabled by defining this macro.

§ PCRE2_CODE_UNIT_WIDTH

#define PCRE2_CODE_UNIT_WIDTH   0

This macro does not have any significance in JPCRE2 context.

It is defined as 0 by default. Defining it before including jpcre2.hpp will override the default (0), but still it will have no effect in a JPCRE2 perspective. Defining it with an invalid value will yield to compile error.
