Detailed Description

template<class Char_T>
struct jpcre2::select32< Char_T >

struct to select 32 bit code unit width and the character type.

If PCRE2_CODE_UNIT_WIDTH 32 is defined you can access it without the suffix (32) i.e jpcre2::select<Char_T>.

Usage: jpcre2::select32<Char_T>.

The character type (Char_T) must be in accordance with the definition of PCRE2_CODE_UNIT_WIDTH. If it doesn't match with the character type, then the behavior is undefined. If the macro JPCRE2_ENABLE_CODE_UNIT_VALIDATION is defined , then such behavior will cause an assertion failure and immediate termination of the program.
