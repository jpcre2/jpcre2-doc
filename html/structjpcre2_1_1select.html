Detailed Description

template<typename Char_T, Ush BS = sizeof( Char_T ) * CHAR_BIT>
struct jpcre2::select< Char_T, BS >

struct to select code unit width and the character type.

Usage: jpcre2::select<Char_T, BS>.

The character type (Char_T) must be in accordance with its' code unit width (BS). If it doesn't match with the character type, compile error will be produced if not suppressed by defining the macro JPCRE2_DISABLE_CODE_UNIT_WIDTH_VALIDATION.

Template Parameters
BSCode unit width (8, 16 or 32)
Char_TCharacter type (char, wchar_t, char16_t, char32_t)

If BS is not given, i.e it is called like

jpcre2::select<Char_T>

then sizeof(Char_T)*CHAR_BIT will be taken as the value for BS.

It is possible to write portable code by using jpcre2::select<Char_T> i.e by not defining the bit size explicitly.

If you want to fix the code unit width for a character type, pass the value as the second templated parameter for the selector. e.g:

jpcre2::select<char, 8>::Regex re;
